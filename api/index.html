Security Settings Web Service API

This WS was created to provide detailes about
our system's users and their security settings.
As a RESTful Serveice all user's detailes will be returned
in JSON format.

Get All Users: [GET]

https://security-settings-ws.herokuapp.com/getAllUsers

Headers:

Debug - Shows indication u have reached the target

JSON Response Example:

[{ "id": 0, "user_name": "un", "password": "pass", "email": "wheresthecake@gmail.com", "first_name": "Moshe", "last_name": "Datz", "settings": { "login_alerts": true, "recognized_devices": ["Macbook Air", "LG G4", "Home PC"], "phone_to_aproove_password_change": "0541111111" } }, { "id": 1, "user_name": "un", "password": "pass", "email": "shauarma@gmail.com", "first_name": "Avi", "last_name": "Biter", "settings": { "login_alerts": true, "recognized_devices": ["Dell inspirion", "IPhone 5s"], "phone_to_aproove_password_change": "0542222222" } }] }

Get User by ID: [GET | POST]

https://security-settings-ws.herokuapp.com/getUserById/###

Parameters:

[GET]
### - user_id : Integer

[POST]
user_id : Integer

Headers:

Debug - Shows indication u have reached the target

Debug-info - Shows the user_id that the service got

JSON Response Example:

[{ "id": 3, "user_name": "un", "password": "pass", "email": "troll@gmail.com", "first_name": "Asi", "last_name": "Cohen", "settings": { "login_alerts": false, "recognized_devices": ["HP Spectre", "Nexus 5"], "phone_to_aproove_password_change": "054333333" } } ]

Get Alerts Devices Number Cross: [POST]

https://security-settings-ws.herokuapp.com/getAlertsDevicesCrossUsers/

Parameters:

[POST]
login_alerts : boolean(true|false) - does the user got login alerts
min_devices : Integer - minimum recognized devices

Headers:

Debug - Shows indication u have reached the target

Debug-info1 - Shows the login_alerts that the service got

Debug-info2 - Shows the min_devices that the service got

JSON Response Example:

[ { "id": 3, "user_name": "un", "password": "pass", "email": "troll@gmail.com", "first_name": "Asi", "last_name": "Cohen", "settings": { "login_alerts": false, "recognized_devices": [ "HP Spectre", "Nexus 5" ], "phone_to_aproove_password_change": "054333333" } } ]

Error Handling

Every requset from the WS can return error JSON or info JSON

Error JSON Example:

{ "error": "error description string" }

Info JSON Example:

{ "info": "info string" }
